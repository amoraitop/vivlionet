# Vivlionet Scraper

Ένα εργαλείο γραμμής εντολών (CLI) γραμμένο σε Python που αναζητά βιβλία στη [Biblionet](https://biblionet.gr), εξάγει πληροφορίες για τις κριτικές/παρουσιάσεις τους και αποθηκεύει τα σχετικά άρθρα ως αρχεία PDF με σκοπό να τα χρησιμοποιήσει / εισάγει στο NotebookLM για παρουσίαση βιβλίων

## Λειτουργικότητα

Το `main.py` εκτελεί τα εξής βήματα:
1. **Αναζήτηση Βιβλίου:** Ζητά από τον χρήστη τον τίτλο ενός βιβλίου και το αναζητά στη βάση δεδομένων της Biblionet.
2. **Εντοπισμός URL:** Βρίσκει τη σελίδα του βιβλίου και πλοηγείται στην ενότητα "Παρουσιάσεις".
3. **Εξαγωγή Δεδομένων:** Συλλέγει πληροφορίες για κάθε κριτική:
   - Συντάκτης
   - Τίτλος άρθρου
   - URL κριτικής
   - Μέσο δημοσίευσης
   - Ημερομηνία
4. **Αποθήκευση σε CSV:** Αποθηκεύει τα παραπάνω δεδομένα σε ένα αρχείο `reviews.csv` (με διαχωριστικό `;`).
5. **Λήψη PDF:** Χρησιμοποιεί το Selenium (Headless Chrome) για να επισκεφθεί τα URL των κριτικών και να τα αποθηκεύσει τοπικά ως αρχεία PDF στον φάκελο `reviews_pdfs/`.

## Προαπαιτούμενα

- Python 3.x
- Google Chrome browser

## Εγκατάσταση

1. Κλωνοποιήστε το αποθετήριο:
   ```bash
   git clone https://github.com/amoraitop/vivlionet.git
   cd vivlionet
   ```

2. Εγκαταστήστε τις απαραίτητες βιβλιοθήκες:
   ```bash
   pip install requests beautifulsoup4 selenium webdriver-manager
   ```

## Χρήση

Τρέξτε το script με τον τίτλο του βιβλίου ως όρισμα ή αφήστε το κενό για να σας ζητηθεί:

```bash
python main.py "Ο τίτλος του βιβλίου"
```
ή
```bash
python main.py
```

Τα αποτελέσματα θα αποθηκευτούν:
- `reviews.csv`: Λίστα με τις κριτικές.
- `reviews_pdfs/`: Φάκελος με τα PDF των κριτικών.
